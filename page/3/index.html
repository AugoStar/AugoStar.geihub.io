AJAX

AJAX = Asynchronous JavaScript and XML(异步的javaScript 和XML).
AJAX不是编程语言，而是一种使用现有标准的新方法。
AJAX最大的优点是在不重新加载整个界面的情况下、可以与服务器交换数据并更新部分网页内容。
AJAX不需要任何浏览器插件，但需要用户允许javascript在浏览器执行
