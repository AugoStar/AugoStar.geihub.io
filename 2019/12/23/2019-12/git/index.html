添加远程主机

git remote add <主机名 origin> <网址>

删除远程主机

git remote rm <主机名>

重命名远程主机

git remote rename <原主机名> <新主机名>

修改远程仓库地址

git remote set-url origin [url]

git fetch <远程主机名>
