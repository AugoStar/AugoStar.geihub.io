python pip 安装模块提示 failed to create process

原因：

python的目录名称或位置发生改变

解决办法：

1：找到python所在目录，打开scripts目录 ，修改python所在的目录 打开scripts目录，找到pip3-script.py文件

2：找到pip-script.py文件 并打开，在第一行替换一下代码 修改成python.exe 路径

pip list 正常显示安装的模块
