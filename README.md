

# 项目信息

vue3 后端ui

需要 Node.js 版本 >= 12.0.0。


### 项目运行

cd [项目名称]

yarn install 

yarn run dev

yarn run dev:example 或者 yarn run dev:exa
** 示例环境**





### 生成依赖

[vue3](https://v3.cn.vuejs.org/)
[UI框架 antdv](https://2x.antdv.com/index-cn)
[vue-router@4 路由](https://next.router.vuejs.org/)


@iconify/vue 图标 


crypto-js 加密
axios ajax请求
pinia@next 存储Store
